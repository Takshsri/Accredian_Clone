# 🚀 Accredian Landing Page Clone

A responsive and modern landing page built using **Next.js, React, and Tailwind CSS**, replicating the Accredian Enterprise website UI and user experience.

---

## 🚀 Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Takshsri/accredian-clone.git
cd accredian-clone

## npm install
## npm run dev
## http://localhost:3000


#🧠 Approach Taken
Built using Next.js App Router
Followed component-based architecture
Each section (Hero, Stats, Clients, Features, FAQ, Testimonials, etc.) is modular
Implemented responsive design using Tailwind CSS
Used Next.js Image optimization for better performance
Created smooth scrolling navigation with active section highlighting (scroll spy)
Designed UI to closely match Accredian’s layout and spacing
